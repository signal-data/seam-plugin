# Seam Plugin — Quick Start

Connect your Seam org:
```
claude mcp add seam --transport http https://mcp.getseam.ai/mcp
```

Try these:
- "Set up an intent spike alert" — get notified when accounts heat up
- "Show my alerts" — list active scheduled alerts
- "Delete my weekly digest" — remove an alert
- "What are my top accounts right now?" — live account signal query
